Redis

概述

  • 解决功能性问题 ：java,jsp,RDBMS,Tomcat,html,linux,jdbc,svn
  • 解决扩展性问题：struts,spring,springMVC,Hibernate,Mybatis
  • 解决性能问题: NoSql,java线程,hadoop,Nginx,MQ,ElasticSearch

简介

NoSql = not only sql “不仅仅是SQL”，泛指非关系型的数据库

不依赖业务逻辑方式存储，而以简单的key-value模式存储

  • 不遵循sql标准
  • 不支持ACID
  • 远超于SQL的性能

场景：高并发，海量数据读写，高拓展

(用sql解决不了，考虑Nosql)

常见Nosql数据库 : Memcache,Redis，MongoDB

  • 行式数据库 （按照每行进行单独存储）
  • 列式数据库

Redis:

  • 开源的key-value存储系统
  • 支持的类型很多，包括string，list，set，zset(sorted set有序集合),hash
  • 都支持push/pop,add/remove以及取交集并集以及更丰富的操作，都是原子性的
  • 支持不同方式的排序
  • 数据都是缓存在内存中
  • redis会周期性的把更新的数据写到磁盘或把修改操作写入追加的记录文件
  • 实现了master-slave(主从)同步

安装

官网

在Linux中

# 需要有gcc编译器，若没有，执行后下面操作
yum install centos-release-scl scl-utils-build
yum install -y devtoolset-8-toolchain
scl enable devtoolset-8 bash
#测试gcc
gcc --version

#将官网下载的解压包放到/opt中
#解压
tar -zxvf xxx.tar.gz
#进入目录，然后执行make命令
#执行完毕，运行make install,最终安装在了/usr/local/bin，进入目录，可看到
cd /usr/local/bin

启动

在前台启动:在对应目录下直接输入：redis-server
