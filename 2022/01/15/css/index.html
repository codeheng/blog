1.1什么是css

Cascading Style Sheet(层叠级联样式表)

css ==> 表现(用于美化网页)

包括：字体，颜色，边距，高度，宽度，背景图片，网页定位，网页浮动…..

CSS预处理器

1.2 发展史

CSS1.0 ==> 只能美化一些基础的

CSS2.0 ==> div(块) + css，提出html与css结构分离的思想，网页变得简单

CSS2.1===> 浮动，定位

CSS3.0 ===> 圆角，阴影，动画…..（需要浏览器兼容性

1.3 快速入门

书写标准格式==>

<!--
    规范:<style>编写css代码，每一个声明，最好用分号结尾
    语法：
        选择器{
            声明1;
            声明2;
            声明3;
        }

此时html和css未分离，在一个文件写的 css在<head>中，html在body中
-->
    <style>
        h1{
            color: aqua;
        }
    </style>

建议分离着写，使用独立于html的css结构，即使用下面规范！

优势:

  1. 内容和表现分离
  2. 网页结构统一，可以实现复用
  3. 样式很丰富
  4. 利于SEO(Search engine optimization)，容易被搜索引擎收录

1.4三种导入方式

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

<!--    2内部样式-->
    <style>
        h1{
            color: yellow;
        }
    </style>
<!--   3，外部样式-->
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
<!-- 1.行内样式-->
    <h1 style="color: red">我是标题</h1>
</body>
</html>

此时显示 行内样式的颜色 若去掉行内样式，则显示 外部样式 对应的颜色 ，因为离h1最近

就近原则 即从上到下，下面覆盖重复上面的样式

外部样式的两种方式：

  • 链接式(目前用这个最多)

    <link rel="stylesheet" href="css/style.css">

  • 导入式(css2.1中使用)

    <style>
    @import "css/style.css";
</style>

2.选择器(重要)

选择页面上的某一个或某一类元素

2.1 基本选择器

标签选择器

会选择到这个页面上的所有标签

类选择器

格式:
.class的名称{   }
好处：可以多个标签归类，是同一个class，可复用
<style>
    .name{
        color: red;
    }
    .id{
        color: yellow;
    }
    .phone{
        color: blue;
    }
</style>

<h1 class="name">name</h1>
<h1 class="phone">phone</h1>
<h1 class="id">id</h1>

id选择器

格式： #id名称{   }
必须全局唯一，不可以复用 
不遵循就近原则  生效原则：id选择器 > 类选择器 > 标签选择器     
<style>
    #test{
        color: blue;
    }
</style>
<h1 id="test">标题1</h1>

2.2高级选择器

1层次选择器

后代选择器

（body 里面所有p都起作用)

body p{
    color: blue;
}

子选择器

只会body里面一代起作用

body>p{
    color: blue;
}

相邻兄弟选择器

(同辈) ===> 向下的相邻

<style>
    .test + p{
    color: blueviolet;
}
</style>

<body>
    <p>p1</p>
    <p class="test">p2</p>
    <p>p3</p>
</body>

此时p3会变颜色

通用选择器

当前选择元素的所有向下的兄弟元素 生效

<style>
    .test~p{
        color: red;
    }
</style>
<body>
    <p class="test">p1</p>
    <p>p2</p>
    <p>p3</p>
</body>

此时p2 p3均为变颜色

2.结构伪类选择器

带冒号这种==> 即伪类 (代表条件判断 )

<body>
    <p>p1</p>
    <p>p2</p>
    <p>p3</p>
    <ul>
        <li>l1</li>
        <li>l2</li>
        <li>l3</li>
    </ul>
</body>
<style>
    /*定位ul中li的第一个与最后一个*/
    ul li:first-child{
        color: red;
    }
    ul li:last-child{
        color: blueviolet;
    }

    /*首先找到当前p元素的父级元素，然后选中父级元素的第一个，并且类型要是当前元素才生效 （按顺序进行选择
    此时p1会变颜色
    */
    p:nth-child(1) {
        color: blue;
    }

    /*选中父元素下的p元素的第二个  此时p2变颜色 (按类型进行选择)  */ 
    p:nth-of-type(2) {
        background-color: blue;
    }
</style>

3.属性选择器(常用)

格式： 标签[id属性] { }

/*选中 a标签 中id属性为 first的元素*/
a[id=first] {  }
/*  选择a标签中class含有 links的元素 
=代表完全匹配 *=只需要含有即可 ^=以什么开头 $= 以什么结尾 (正则表达式)
*/
a[class *= "links"] { }

            margin: 0;
            padding: 0;

        }


     border: 1px solid red;

5.浮动

标准文档流

块级元素：独占一行

h1~h6, p标签 div标签 列表

行内元素：不独占一行

span  a标签 img strong....

display 也是实现行内元素排列的方式，但多用float ===> 方向不可以控制，不会出现父级边框塌陷问题

block 块元素
inline 行内元素
inline-block 是块元素，但是可以内联，在一行

none ==> 消失

float浮动() ===> 浮动的话可能会脱离标准文档流，所以要解决父级边框塌陷

right,left 左右浮动

clear:both; /*两侧均不许有浮动元素，即按照正常元素向下排布*/
clear:right;
clear:left;
clear:none; /*不清理*/

浮动会出现父级边框塌陷问题

如何解决？

1.将父级元素(div)边框高度变大

#father{
    border:1px #000 solid;
    height:800px;
}

2.再上面div后加一个空的div，然后清除它的浮动 （尽量避免空div，不用此方法）

<div class = "clear"></div>
.clear{
    clear:both;
    margin:0;
    padding:0;
}

3.利用overflow

/*设置父级元素overflow  根据里面内容而进行变化*/
#father{
    border:1px #000 solid;
    overflow:hidden; /*网页若下拉的场景不能使用*/
}

4.在父类中添加一个伪类**(推荐使用)**

#father{
    border:1px #000 solid;
}
#father:after{
    conten:"";
    display:block;
    clear:both;
}

相对定位

相对于原来的位置进行指定偏移，位置仍然在标准文档流中，原来位置被保留

position:relative;  /* 然后设置 距离 上下左右多少*/
top:
bottom:
left:
right:

<style>
        #box{
            height: 300px;
            width: 300px;
            border: 1px solid red;
            padding: 10px;
        }
        a{
            text-decoration: none;
            height: 100px;
            width: 100px;
            background-color: pink;
            line-height: 100px;
            text-align: center;
            color: white;
            display: block;
        }
        a:hover{
            background-color: cornflowerblue;
        }
        .a2,.a4{
            position: relative;
            left: 200px;
            top: -100px;
        }
        .a5{
            position: relative;
            left: 100px;
            bottom: 300px;
        }
</style>
<div id = "box">
    <a href="#" class="a1">链接1</a>
    <a href="#" class="a2">链接2</a>
    <a href="#" class="a3">链接3</a>
    <a href="#" class="a4">链接4</a>
    <a href="#" class="a5">链接5</a>
</div>

绝对定位和固定定位

position: absolute;
position: fixed;
<style>
    body{
        height: 1000px; /*会出现滚动条  当滚动时，固定定位一直不变，而绝对定位只是对于浏览器右边不变，上下会变*/
    }
    /*绝对定位，相对于浏览器*/
    div:nth-of-type(1) {
        position: absolute;
        right: 0;
        bottom: 0;
        width: 100px;
        height: 100px;
        background: red;
    }
    /*固定定位*/
    div:nth-of-type(2) {
        position: fixed;
        right: 0;
        bottom: 0;
        width: 50px;
        height: 50px;
        background: blue;
    }
</style>
