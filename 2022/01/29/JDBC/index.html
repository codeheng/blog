1.JDBC简介

​ Java DataBase Connectivity（Java语言连接数据库）

JDBC的本质是什么？

  • JDBC是SUN公司制定的一套接口（interface)

    在包 java.sql.*;中

接口都有调用者和实现者。
面向接口调用、面向接口写实现类，这都属于面向接口编程。

为什么要面向接口编程？
解耦合：降低程序的耦合度，提高程序的扩展力。
多态机制就是非常典型的：面向抽象编程。（不要面向具体编程）

建议：
    Animal a = new Cat();
    Animal a = new Dog();
// 喂养的方法
public void feed(Animal a){ // 面向父类型编程。
}
不建议：
    Dog d = new Dog();
    Cat c = new Cat();

思考：为什么SUN制定一套JDBC接口呢？
    因为每一个数据库的底层实现原理都不一样。
    Oracle数据库有自己的原理。
    MySQL数据库也有自己的原理。
    MS SqlServer数据库也有自己的原理。
    ....
    每一个数据库产品都有自己独特的实现原理。

JDBC的本质到底是什么？
    一套接口。

2.JDBC六步

（背会）

第一步：注册驱动（作用：告诉Java程序，即将要连接的是哪个品牌的数据库）

第二步：获取连接（表示JVM的进程和数据库进程之间的通道打开了，这属于进程之间的通信，重量级的，使用完之后一定要关闭通道。）

第三步：获取数据库操作对象（专门执行sql语句的对象）

第四步：执行SQL语句（DQL DML….）

第五步：处理查询结果集（只有当第四步执行的是select语句的时候，才有这第五步处理查询结果集。）

第六步：释放资源（使用完资源之后一定要关闭资源。Java和数据库属于进程间的通信，开启之后一定要关闭。）

示例

package com.heng.jdbc;
import java.sql.*;
public class JDBCtest01 {
    public static void main(String[] args) {
        Connection connection = null;
        Statement statement = null;
        try {
            //1.注册驱动
            Driver driver = new com.mysql.jdbc.Driver();
            DriverManager.registerDriver(driver);

            //2.获取连接(需要知道url,用户名,密码)
            String url = "jdbc:mysql://localhost:3306/test";
            String username = "root";
            String pwd = "123";
            connection = DriverManager.getConnection(url,username,pwd);

            System.out.println("数据库对象" + connection);//打印查看sql对象 
            //结果：数据库对象com.mysql.cj.jdbc.ConnectionImpl@587c290d

            //3.获取sql对象
            statement = connection.createStatement();

            //4执行sql语句
            String sql = "insert into class(class_no,class_name) value('5','数学班')";
            //执行,为(insert,delete,update)提供的方法
            //返回值是 影响数据库中的记录条数
            int count = statement.executeUpdate(sql);
            System.out.println( count == 1 ? "sql success!" : "Not！");
               //5.处理sql结果集，因为没有查询，

        } catch (SQLException e) {
            e.printStackTrace();
        }finally {
            /*6.关闭连接
                遵循从大到小的依次关闭
                connection开启的statement,所以statement先关闭
                分别try..catch
            * */
            if (statement != null) {
                try {
                    statement.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (connection != null) {
                try {
                    connection.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
        }

    }
}

import java.sql.*;

            e.printStackTrace();
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        } finally {//6
            if (state != null) {
                try {
                    state.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (conn != null) {
                try {
                    conn.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }

        }
    }

完成查询

public class JdbcExec {
    public static void main(String[] args) {
        Connection conn = null;
        Statement state = null;
        ResultSet rs = null; //查询出来的结果集合
        try {
            //1.注册驱动
            Class.forName("com.mysql.jdbc.Driver");
            //2.获取连接
            conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/test","root","123");
            //3.获取sql对象
            state = conn.createStatement();
            //4执行sql语句，此时执行查询语句
            String sql = "select *from class";
            rs = state.executeQuery(sql);
            //5.处理sql结果集
            while ( rs.next() != false) {
                //next()相当于光标，持续下移，若有数据返回true
                //jdbc中下标中1开始，此时表示取光标第一行第一列 也可以用 最终查询表的列名(推荐)
                //String no = rs.getString(1);
                String no = rs.getString("class_no");
                String name = rs.getString("class_name");//第一行第二列
                System.out.println(no + "," + name);
            }
        } catch (SQLException e) {
            e.printStackTrace();
        } catch (ClassNotFoundException e) {
            e.printStackTrace();
        } 
        //释放资源，先查询结果集==>再数据库操作对象===>最后连接
        finally {
            if (rs != null) {
                try {
                        rs.close();
                    } catch (SQLException e) {
                        e.printStackTrace();
                }
            }
            if (state != null) {
                try {
                    state.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (conn != null) {
                try {
                    conn.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }

        }
    }
}

3.示例

简单模仿登陆

1让用户输入 账号密码

2 和sql中的数据，进行比较，若存在相同，则登陆成功，否则失败

创建表

编写JdbcLogin.java

public class JdbcLogin {
    public static void main(String[] args) {
        //1提供用户输入,初始化 返回 账号和密码
        Map<String,String> rs = init();
        System.out.println(rs);

        //2.jdbc操作判断是否相等
        System.out.println(login(rs) == true ? "登陆成功" : "失败!");
    }

    private static boolean login(Map<String, String> rs) {
        Connection conn = null;
        Statement statement = null;
        ResultSet result = null;
        boolean flagSuccess = false; //标志是否登陆成功
        try {
            //1 注册驱动
            Class.forName("com.mysql.jdbc.Driver");
            //2 获取连接
            conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/user","root","123");
            //3获得sql对象
            statement = conn.createStatement();
            //4执行sql 
            String sql = "select name,password from t_user where name='"+rs.get("inputName")+"' " +
                    "and password='"+rs.get("inputPwd")+"'   ";
            //5 处理结果集
            result = statement.executeQuery(sql);
            if ( result.next() != false ) {
                flagSuccess = true;//登陆成功！
            }

        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            //6 释放资源
            if (result !=null) {
                        try {
                            result.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (statement !=null) {
                try {
                    statement.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (conn !=null) {
                try {
                    conn.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
        }
        return flagSuccess;
    }

    private static Map<String, String> init() {
        Scanner s = new Scanner(System.in);
        System.out.print("用户名：");
        String inputName = s.nextLine(); //获得输入，为字符串
        System.out.print("密码：");
        String inputPwd = s.nextLine();
        HashMap<String, String> map = new HashMap<>();
        map.put("inputName",inputName); //组装成map(key,value)
        map.put("inputPwd",inputPwd);

        return map;
    }
}

经过测试，当输入数据库存在的数据时，均登陆成功，但存在bug，会导致**SQL注入 ** 如图

此时输入qza' or '1'='1 则会将or参与到sql语句中，导致sql语句查询恒为真 如图debug查看

注入原因：输入的信息含sql关键字，并且这些关键字参与到了sql的编译，导致原意扭曲

如何解决：

只要提供的信息，不参与sql语句编译即可！

利用java.sql.preparedStatement 此接口继承了java.sql.Statement

属于预编译的数据库操作对象 (预先对sql语句的框架进行编译，然后再给sql传值)

改进如下

private static boolean login(Map<String, String> rs) {
        Connection conn = null;
        PreparedStatement ps = null; //采用预编译对象
        ResultSet result = null;
        boolean flagSuccess = false;
        String name = rs.get("inputName");// 获取输入的用户和密码
        String pwd = rs.get("inputPwd");
        try {
            //1 注册驱动
            Class.forName("com.mysql.jdbc.Driver");
            //2 获取连接
            conn = DriverManager.getConnection("jdbc:mysql://localhost:3306/user","root","123");
            //3获得sql对象 ?代表占位符，用来接收一个值
            String sql = "select name,password from t_user where name=? and password=?";
            ps = conn.prepareStatement(sql);
            //给问号传值，第一个?下标索引1...
            ps.setString(1,name);
            ps.setString(2,pwd);
            //4执行sql
            result = ps.executeQuery();
            //5 处理结果集
            if ( result.next() != false ) {
                //登陆成功！
                flagSuccess = true;
            }
        } catch (Exception e) {
            e.printStackTrace();
        } finally {
            if (result !=null) {
                try {
                    result.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (ps !=null) {
                try {
                    ps.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
            if (conn !=null) {
                try {
                    conn.close();
                } catch (SQLException e) {
                    e.printStackTrace();
                }
            }
        }
        return flagSuccess;
    }

总结：

Statement存在sql注入问题，并且是编译一次执行一次，因为sql语句根据每次用户输入，都会变化

PreparedStatement解决了sql注入问题，编译一次，可执行N次，效率更好，而且在编译阶段会做安全检查

故大多数用PreparedStatement，除非业务方面需要sql注入进行拼接，才使用Statement,比如商城按照xx排序即order by desc，则此是需要sql注入

Jdbc01.java

package com.heng.jdbc;
import java.sql.*;
public class Jdbc01 {
    public static void main(String[] args) throws ClassNotFoundException, SQLException {
        //配置信息
        String url = "jdbc:mysql://localhost:3306/user?useUnicode=true&characterEncoding=utf-8";
        String username = "root";
        String password = "123";
        //1驱动
        Class.forName("com.mysql.jdbc.Driver");
        //2连接数据库
        Connection connection = DriverManager.getConnection(url, username, password);
        //3获得sql对象
        Statement statement = connection.createStatement();
        //4执行sql
        ResultSet rs = statement.executeQuery("select *from t_user ");
        //5结果集
        while ( rs.next() ) {
            System.out.println("name = "+ rs.getObject("name"));
            System.out.println("password =" + rs.getObject("password"));
        }
        //6断开连接,释放资源   
        rs.close();
        statement.close();
        connection.close();
    }
}
