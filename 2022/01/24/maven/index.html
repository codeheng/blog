10. pom.xml文件

<?xml version="1.0" encoding="UTF-8"?>
<!--maven版本和头文件-->
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"

<!--  下面这是我们配置的GAV-->
  <groupId>org.example</groupId>
  <artifactId>untitled</artifactId>
  <version>1.0-SNAPSHOT</version>
<!--  package 项目的打包方式
jar：java应用
war：javaweb应用

  <!-- FIXME change it to the project's website -->

  <properties>
<!--    项目的默认构建编码-->

maven仓库

只需加上dependency。在maven仓库中，选择自己需要的即可，maven将自己导入

由于maven 约定大于配置 的规定，所以再之后写的配置文件，会出现无法导出或生效的问题，解决方法：

在mybatis 会遇到

<!--在build中配置resources,来防止我们资源导出失败的问题-->
<build>
    <resources>
      <resource>
        <directory>src/main/resources</directory>
        <includes>
          <include>**/*.properties</include>
          <include>**/*.xml</include>
        </includes>
        <filtering>true</filtering>
      </resource>
      <resource>
        <directory>src/main/java</directory>
        <includes>
          <include>**/*.properties</include>
          <include>**/*.xml</include>
        </includes>
        <filtering>false</filtering>
      </resource>
    </resources>
  </build>

若每次都需要重新配置，修改全局配置

若maven配置完之后，创建项目出现Error reading file E:/idea_project/Maven_pro/pom.xml

重新安装maven，然后配置环境，则可进行解决

11.简单编写HelloServlet

maven app结构

HelloServlet.java中

public class HelloServlet extends HttpServlet {
    //重写doGet 和 doPost方法
    @Override
    protected void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException,IOException {
        response.setContentType("text/html");
        response.setCharacterEncoding("utf-8");  //解决中文乱码！
        PrintWriter out = response.getWriter();
        out.println("<html>");
        out.println("<head>");
        out.println("<title>你好</title>");
        out.println("</head>");
        out.println("<body>");
        out.println("<h1>你好</h1>");
        out.println("</body>");
        out.println("</html>");
    }
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

<!DOCTYPE web-app PUBLIC
 "-//Sun Microsystems, Inc.//DTD Web Application 2.3//EN"
 "http://java.sun.com/dtd/web-app_2_3.dtd" >

<web-app>
  <display-name>Archetype Created Web Application</display-name>

  <!--  web.xml是web的核心应用
注册servlet-->
  <servlet>
    <servlet-name>helloServlet</servlet-name>
    <servlet-class>com.heng.servlet.HelloServlet</servlet-class>
  </servlet>
  <!--  一个servlet对应一个mapping 映射-->
  <servlet-mapping>
    <servlet-name>helloServlet</servlet-name>
    <!--    映射请求路径-->
    <url-pattern>/heng</url-pattern>
  </servlet-mapping>

</web-app>
