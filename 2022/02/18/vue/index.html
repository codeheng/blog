vue

简介

官网https://vuejs.bootcss.com/guide/

  • JavaScript框架
  • 简化Dom操作
  • 响应式数据驱动

七大属性

  • el属性

    用来指示vue编译器从什么地方开始解析 vue的语法，可以说是一个占位符。

  • data属性

    用来组织从view中抽象出来的属性，可以说将视图的数据抽象出来存放在data中。

  • template属性

    用来设置模板，会替换页面元素，包括占位符。

  • methods属性

    放置页面中的业务逻辑，js方法一般都放置在methods中

  • render属性

    创建真正的Virtual Dom

  • computed属性

    用来计算

  • watch属性

    • watch:function(new,old){}

    • 监听data中数据的变化

    • 两个参数，一个返回新值，一个返回旧值，

Vue指令

1、v-text 设置标签的内容

默认会替换全部内容，使用{{}} 可替换指定内容

<h2 v-text="message + 'haha' "> </h2>

页面将显示出你好 Vue!haha

2、v-html 设置标签的innerHTML

<p v-html="content"></p>

data: {
    content: "<a href='http://www.baidu.com'>百度</a>"
}

此时将出现百度的超链接，点击则跳转

3、v-on 为元素绑定事件

<input type="button" value="v-on指令" v-on:click="doIt">
<input type="button" value="v-on简写" @click="doIt">
<h1 @click="changeData">{{food}}</h1>

<script>
    var app = new Vue({
        el: "#app",
        data: {
            food: "土豆"
        },
        methods: {
            doIt: function() {
                alert("hhhhhhhhhh") //弹窗
            },
            changeData: function() {
                this.food += "好吃"  //点击文字土豆，然后会拼接字符串
            }
        }
    })
</script>

4、v-show 根据表达式真假，切换元素的显示和隐藏

原理是修改元素的display 

<img src="http://img.duoziwang.com/2021/01/1618792249721642.jpg" v-show="isShow" alt="小图像">
<input type="button" value="切换是否显示图片" @click="changeIsShow">
<script>
    var app = new Vue({
        el: "#app",
        data: {
            isShow: false
        },
        methods: {
            changeIsShow: function() {
                this.isShow = !this.isShow
            }
        }
    })
</script>

5、v-if 根据表达式真假切换元素的显示状态

本质是操作dom元素来切换显示状态

若为true，元素存在dom树中，为false，从dom树中移除

<p v-if="isShow1">come on!!</p>
<input type="button" value="点我切换" @click="toggleIsShow">
isShow1: false
toggleIsShow: function() {
    this.isShow1 = !this.isShow1
}

6、v-bind 设置元素的属性 ，比如(src,title,class)

完整写法: v-bind:属性名 v-bind可以省略

<style>
    .active {
        border: 1px solid red;
    }
</style>
<img :src="imgSrc" alt="" :title="imgTitle" :class="{active:isActive}" @click="toggleActive ">
<script>
    var app = new Vue({
        el: "#app ",
        data: {
            imgSrc: "http://www.itheima.com/images/logo.png",
            imgTitle: "我是黑马！！",
            isActive: false,
        },
        methods: {

            toggleActive: function() {
                this.isActive = !this.isActive
            }
        }
    })
</script>

当点击图片时，class样式被激活，此时显示出来外边框
