acwing-小白鼠排队

题目链接 (北京大学考研机试题)

简介

$N$只小白鼠，每只鼠头上戴着一顶有颜色的帽子，现在称出每只白鼠的重量，要求按照白鼠重量从大到小的顺序输出它们头上帽子的颜色 （帽子的颜色用 red，blue 等字符串来表示。）

不同小白鼠可以戴相同颜色的帽子，白鼠重量用整数表示

输入格式

第一行： 输入$N$，表示白鼠的数量

下面$N$行中，第一个输入 <= 100的正整数，表示白鼠重量；第二个为字符串(不超过10字符)，表示帽子颜色

输出格式

按照白鼠的重量从大到小的顺序输出白鼠的帽子颜色

数据范围

$1 \le N \le 100$

输入样例：

3
30 red
50 blue
40 green

输出样例：

blue
green
red

分析

将输入的两者存起来，然后根据重量比较大小，然后大的将其对应的颜色输出*

用什么数据结构存 ？ C++中的pair

C++中的 pair 是一种元组 (tuple),它包含两个值，分别代表键和值

定义如下：

template <typename K, typename V>  
// first 成员为默认值，second 成员为默认值
struct pair {  
    K first_;  // K 表示键类型
    V second_;  // V 表示值类型
};

可以使用以下语法来创建 pair 对象:

pair<Key, Value> p1(key1, value1);  
pair<Key, Value> p2(key2, value2);

pair 类型还支持迭代器访问和遍历。例如:

for (const auto& p : pairs) {  
    std::cout << "Key: " << p.first << ", Value: " << p.second << std::endl; 
}

知道了pair的用法，那么这个题就迎刃而解

解答

#include <iostream>
#include <algorithm>

using namespace std;

const int N = 110;

int main()
{
    // 定义pair
    pair<int, string> q[N];
    
    int n;
    cin >> n;
    // 输入到pair中
    for (int i = 0; i < n; i ++)
        cin >> q[i].first  >> q[i].second;
    // 排序,sort默认是从小到大
    sort(q, q + n);
    
    // 输出对应的value
    for (int i = n - 1; i >= 0; i --)
        cout << q[i].second << endl;
    
    return 0;
}

关于sort 默认是从小到大,是C++中STL提供的排序函数

sort (first, last) ，对容器或数组中的[first, last) 范围元素排序

或要改为从大到小，则sort(q, q+n, greater<>())
