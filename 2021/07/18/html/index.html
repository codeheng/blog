1、概述

Hyper Text Markup Language(超文本标记语言)

W3C = World Wide Web Consortium（万维网联盟)

万维网联盟

标准： 结构化标准语言(html,xml)、表现标准语言(css)、行为标准(dom，ECMASript)

利用intelliJ IDEA 进行开发 如图

HTML是负责描述文档语义的语言

网页基本信息

<!--我是注释 -->
<!--DOCTYPE 告诉浏览器，使用什么规范 -->
<!DOCTYPE html>
<html lang="en">
<!--head网页头部-->
<head>
<!--meta 描述性信息 ，一般用于做SEO=搜索引擎优化-->
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<!--body网页主体-->
<body>

</body>
</html>

5.表单

<form method="post" action="result.html">
    <p> 名字：<input name = "username " value="Yiheng"  type="text"></p>
    <p> 密码：<input name = "pwd " type="password"></p>
    <p>
        <input type= "submit" name = "Button" value="提交">
        <input type="reset" name = "Reset" value = "重填">
    </p>
</form>

  • method 请求方式==> get/post

    • get： 可以在url中看到提交的信息，不安全但高效
    • post：比较安全，可以传输大文件

  • action 表单提交的位置，可以是网站，也可以是一个请求处理地址

  • value ==> 表示该文本框的默认初始值

    表单格式

单选框radio

<p> 性别:
    <input type="radio" value="girl" name = "sex" />女
    <input type="radio" value="boy" name = "sex"/>男
</p>

radio : 单选框标签

name 表示组，要这俩是同一个组，不可以进行同时选择

多选框 checkbox

<p> 爱好：
       <input type="checkbox" value = "sing" name="hobby">唱
       <input type="checkbox" value = "dance" name="hobby">跳
       <input type="checkbox" value = "rap" name="hobby">rap
       <input type="checkbox" value = "basketball" name="hobby">篮球
</p>

按钮 button

<p>按钮:

type:

  • button ==> 普通按钮
  • image ==> 图像按钮

  • reset ==> 重置
